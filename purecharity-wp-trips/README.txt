== Description ==

Plugin to display Pure Charity Trips on your wordpress install via shortcodes or template tags. There are two shortcodes available and six template tags ready for use.

== Installation ==

1. Upload the `purecharity-wp-trips` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You're done!
